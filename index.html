Welcome to the VGP GenomeArk! This site provides access to genome sequencing and assembly data generated by the Vertebrate Genomes Project. Four genomic data types are available for most species, including long reads, linked reads, optical maps, and Hi-C reads. The draft assemblies incorporating these data are expertly curated before submission to NCBI and EBI for annotation and archival. The links below provide summary assembly statistics and instructions for downloading the raw data and assemblies. Expert AWS users can also access this data directly from s3://genomeark and more technical information is available from https://github.com/vgp.
